## Beginner instructions

1. Unzip this folder.
2. Upload the files inside this project folder to a new GitHub repository.
3. Go to Vercel.
4. Click New Project.
5. Import your GitHub repository.
6. Click Deploy.

## Optional local test

If you have Node.js installed:

npm install
npm run dev

Then open http://localhost:3000
